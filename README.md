# Talha Rashid Portfolio

A bold one-page portfolio for Talha Rashid that highlights recent AI, data, web, and MLOps projects with a stronger visual presentation.

## Live site

[https://rtalhaa.github.io/portfolio/](https://rtalhaa.github.io/portfolio/)

## Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages via GitHub Actions

## Local preview

Run a simple static server from the project root:

```powershell
python -m http.server 4173
```

Then open [http://localhost:4173](http://localhost:4173).
